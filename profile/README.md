## About · 关于

Gamebunensi is a home for small, self-contained browser games and interactive
web experiments created by [Anbunensi](https://github.com/Anbunensi). Each
project can be opened and played directly in a modern browser.

Gamebunensi 收录由 [Anbunensi](https://github.com/Anbunensi) 制作的轻量网页
游戏与交互实验；项目无需安装，使用现代浏览器即可体验。
