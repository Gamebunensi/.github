## 关于 · About

Gamebunensi 是 [Anbunensi](https://github.com/Anbunensi) 的网页创意实验室。这里收录轻量小游戏、语言工具和互动视觉作品——无需下载客户端，使用现代浏览器即可直接体验。

Gamebunensi is a collection of small, self-contained browser experiences created by [Anbunensi](https://github.com/Anbunensi). Every project is designed to be easy to open, understand and play with.

### 我们在做什么

- **规则小游戏**：用不断变化的限制制造意外、幽默和挑战。
- **语言小工具**：探索中文与英文之间的读音、谐音和表达方式。
- **互动实验**：让 Canvas、动画与输入反馈变成轻巧的网页体验。

### 体验原则

- 打开浏览器就能使用，不设置不必要的登录与安装步骤。
- 优先适配桌面端和移动端，鼠标与触屏都能操作。
- 能在本地完成的计算和记录，尽量留在用户的浏览器中。
- 每个项目只聚焦一个清楚、有趣的核心想法。
